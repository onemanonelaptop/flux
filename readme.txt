== Installation ==
1. Verify that you have PHP5, which is required for this plugin.
2. Upload the whole `flux` directory to the `/wp-content/plugins/` directory or choose Plugins -> Add New from your Dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress
